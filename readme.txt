Steps to run assegment:-
	1. Install nodeJs and visual code.
	2. open visual code.
	3. open terminal in linux and Command promt in windows.
	4. run command: npx create-react-app my-app.
	5. replace given file from my-app directory with file given above.
	6. now run command: cd my-app.
	7. then run command: npm start.
	9. wait for browser to open the given react app.
